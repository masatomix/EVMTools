PV (Planned Value) 予定のスケジュール。現時点で、どれだけ成果(工数)をださなくてはいけないかの値。
EV (Earned Value) 実績の値。現時点で、どれだけ成果(工数)をだせているかの値。
AC (Actual Cost) 実績の値。現時点で、どれだけ工数を使用したかの値。
SPI (Schedule Performance Index) スケジュール効率指数 := EV/PV
　　1.0 より大きければ、予定のスケジュール(PV)より、進捗(EV)が進んでいることを表します。
CPI (Cost Performance Index) コスト効率指数 := EV/AC
　　1.0 より大きければ、よりすくない稼働(AC)で、効率よく成果(EV)をだせていることを表します。

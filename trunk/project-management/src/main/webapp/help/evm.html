BAC (Budget at Completion) 完了時の総工数
SV (Scheduled Variance) スケジュールの差異 := EV-PV
CV (Cost Variance) コストの差異 := EV-AC
SPI (Schedule Performance Index) スケジュール効率指数 := EV/PV
CPI (Cost Performance Index) コスト効率指数 := EV/AC
ETC (Estimate To Completion) 残作業コスト予測 := (BAC-EV)/CPI
EAC (Estimate At Completion) 完了時コスト予測 := AC+ETC
VAC (Variance At Completion) 完了時コスト差異 := BAC-EAC
